BookSky :

BookSky is a simple and responsive web application that allows users to manage and organize their book collection. Users can add book details and view them dynamically through an intuitive interface.

Features :

* Add new books with title, author, and description
* Display books dynamically on the webpage
* User-friendly and responsive design
* Simple and lightweight implementation

 Technologies Used:

* HTML5
* CSS3
* JavaScript

Project Structure


BookSky/
│
├── index.html
├── style.css
└── script.js


Getting Started :

1. Clone the repository:

git clone https://github.com/divya408/booksky-app.git

2. Open the project folder.

3. Run `index.html` in your browser.
